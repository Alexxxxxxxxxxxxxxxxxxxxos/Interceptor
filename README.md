# INSTALLATION(LINUX):
```bash
git clone https://github.com/Lucksi/Interceptor
cd Interceptor
sudo chmod +x Installer.sh
sudo ./Installer.sh
```

<br>

# USAGE(LINUX):
```bash
sudo ./Interceptor.sh
```

<br>

# IF YOUR LINK DOESN'T SHOW UP TRY TO WRITE THIS COMMAND:
```bash
link=$(curl -s -N http://127.0.0.1:4040/api/tunnels | grep -o "https://[0-9a-z]*\.ngrok.io")
```
